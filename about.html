ABOUT

Archive Assistant is a transformative fan utility tool created for fans. This project is built upon a foundation of deep respect for the Organization for Transformative Works (OTW) and the Archive of Our Own (AO3). Its mission is to provide a tool that enhances the personal organization and preservation of fanworks without compromising the integrity of the Archive or the rights of its creators

This tool is intended for personal use only and is intended to function as a companion and not a replacement for (AO3) the Archive of Our Own.

Privacy Notice

Last Updated: February 2024

1. Local Processing

  1. Archive Assistant does not have a central database. All file parsing (EPUB/PDF/HTML) and metadata extraction happen locally within the browser.
  2. localStorage is used to save recent "History" and "Extraction Jobs." This data never leaves the localstorage of the device. this tool will not store any data for the purpose of transferring or accessing from another browser.
  3. Temporary variables used during the extraction process may be cleared when you refresh the page or close the tab.

2. External Connections

  1. the "paste from AO3" feature sends a request to the Archive of Our Own servers to retrieve publicly available metadata. Archive assistant does not retrieve your personal AO3 account details or login credentials.
  2. This tool loads aesthetic fonts from Google Fonts. This may involve minimal logging of your IP address by Google for the sole purpose of serving the font files.
  3. This tool uses an external api server to request works from ao3.

3. Cookies & Tracking

  1. no use of Google Analytics, Facebook Pixels, or any other third-party tracking scripts for data tracking and data analysis.
  2. This does not use traditional tracking cookies. The Banner upon entry is a consent notice for the use of Local Storage, which your browser uses to remember themes, preferences, and upload history.

4. Data and privacy

  1. the user has full control over data by clearing individual history entries, using the "clear all" functions in the settings to instantly wipe all stored information from the browser.
  2. no data is stored in cloud or other file hosting servers. Data is only saved when it is externally copied or downloaded by using the download csv and copy tsv features to export extracted data.

Fair use of backend API server

  1. By Entering an AO3 URL, the tool sends the link to an API server which requests the HTML from an AO3 work. It does not use your cookies, session data, or Ao3 login credentials.
  2. None of the tools or servers used to host this page store any data it handles in a database. It does not keep logs of URLs, history of searches. The script reads the metadata from the AO3 HTML and packages it into a JSON object. This json is sent back to the browser. Once the data reaches your device, it is handled according to the Local Storage policy, and no copy of the metadata remains online.
